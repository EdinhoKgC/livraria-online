Python Shell - Administração

Aviso de Segurança

Esta ferramenta permite executar código Python real com contexto persistente. Use com responsabilidade e apenas para tarefas administrativas legítimas. Todos os comandos são registrados e monitorados.

Sintaxe Correta para Imports:
  • from catalogo.models import Livro ✅
  • from catalogo import models ✅
  • import catalogo.models ✅
  • from catalogo import models.Livro ❌ (sintaxe incorreta)
Como usar:
  • Digite código Python como se fosse um arquivo .py
  • Use import para importar módulos e modelos
  • O contexto é persistente entre comandos
  • Use print() para exibir resultados
  • Variáveis ficam disponíveis para próximos comandos
  • Tab: Indentação automática (4 espaços)
  • Enter: Nova linha com indentação inteligente
  • Ctrl+Enter: Executar código
